## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → Route Handler /api/cobranca/nfse | Valor e arrays controlados pelo operador — validados no Route Handler antes de chamar backend |
| Browser → Route Handler /api/athos/venda/[idvenda]/tipo-produto | Parâmetro idvenda da URL — validado como inteiro positivo antes de chamar backend |
| Route Handler → Backend NestJS | Comunicação server-to-server via backendFetch com x-internal-api-key injetado automaticamente |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
