    Query SQL a usar (D-04, validada no banco):
    ```
    SELECT
      BOOL_OR(p.tipoproduto) as tem_produto_fisico,
      BOOL_AND(NOT COALESCE(p.tipoproduto, false)) as todos_servico
    FROM venda_item vi
    JOIN produto p ON p.idproduto = vi.idproduto
    WHERE vi.idvenda = $1 AND COALESCE(vi.cancelada, false) = false
    ```

    Tratamento do resultado: `const row = result.rows[0]`; retornar `{ temProdutoFisico: row?.tem_produto_fisico ?? false, todosServico: row?.todos_servico ?? true }` — o `?? false` e `?? true` cobrem o caso NULL (Pitfall 4 do RESEARCH.md).
