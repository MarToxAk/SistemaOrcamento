## Deviations from Plan

### RULE 2 — Funcionalidade critica faltante: preview da galeria impossivel

- **Found during:** Task 2 (botao "Pre-visualizar" dos itens da galeria)
- **Issue:** O backend (Plano 05) nao expunha o `source` de um template salvo: `GET /pdf-templates` retorna `PdfTemplateSummary` (sem `source`) e `POST /pdf-templates/preview` so aceitava `source` explicito. Isso tornava impossivel pre-visualizar um preset/upload ja salvo sem o admin colar o conteudo de novo — quebrando D-08 para o fluxo de galeria.
- **Fix:** Adicionado `PdfTemplatesService.getSource(id)` e estendido o controller para aceitar `templateId` opcional no `preview`, resolvendo o `source` via `getSource` antes de chamar `renderPreviewPdf` (mesmo caminho hardened, T-999.1-17). Retorna 400 se nem `source` nem `templateId` forem informados.
- **Files modified:** pdf-templates.service.ts, pdf-templates.controller.ts, pdf-templates.controller.test.ts (2 testes novos)
- **Commit:** d77d68c

### RULE 4 — Mudanca arquitetural aprovada pelo usuario: gate de senha na UI

- **Found during:** Checkpoint humano (usuario perguntou se qualquer um poderia mudar o layout)
- **Issue:** O app nao tem login/perfis (D-03 partiu de "deploy interno"). A `ADMIN_API_KEY` protege o backend, mas a propria tela `/configuracoes/templates` nao exigia credencial — qualquer pessoa que chegasse na URL poderia trocar o layout.
- **Decisao:** Apresentadas 3 opcoes ao usuario (prompt de senha / Basic Auth na rota / restricao por IP). Usuario escolheu (a) prompt de senha.
- **Fix:** `admin-session.ts` (cookie httpOnly assinado por HMAC, validade 8h, `timingSafeEqual`), rota `/api/admin/login` (GET status / POST valida `CONFIG_PANEL_PASSWORD` server-side e seta cookie / DELETE logout), `requireAdminSession` aplicado nas 4 rotas proxy, e tela de senha + botao "Sair" no manager. **Falha fechada:** sem `CONFIG_PANEL_PASSWORD` o gate fica inativo (preserva o modelo interno original D-03); com ela definida, a tela exige autenticacao.
- **Files modified:** admin-session.ts (novo), api/admin/login/route.ts (novo), 4 rotas proxy, templates-manager.tsx
- **Commit:** f521344
