|----------|-------------|
| template ativo (storage) -> Handlebars | source pode ter vindo de upload de usuario |
| HTML renderizado -> Chromium headless | renderizar HTML arbitrario = executar codigo nao confiavel |
| Chromium -> rede/host | tentativa de SSRF/exfiltracao/leitura de arquivo |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-999.1-07 | Elevation of Privilege | Handlebars.compile de template nao confiavel | mitigate | knownHelpersOnly:true; sem helpers de I/O; allowCallsToHelperMissing=false; Handlebars.create() isolado (Pattern 1, Pitfall 3) |
| T-999.1-08 | Information Disclosure | SSRF via Puppeteer (metadata 169.254.169.254, IPs internos) | mitigate | setRequestInterception abortando toda rede exceto inline (Pattern 2) |
| T-999.1-09 | Information Disclosure | exfiltracao via img/beacon externo | mitigate | mesmo bloqueio de rede |
| T-999.1-10 | Information Disclosure | leitura de arquivo local file:// / @import | mitigate | abort de file:// na interceptacao |
| T-999.1-11 | Denial of Service | template gigante / loop de render | mitigate | timeout 30s no setContent; Throttler global; limite de tamanho aplicado no upload (Plano 05) |
| T-999.1-SANDBOX | Elevation of Privilege | Chromium root + --no-sandbox com HTML nao confiavel | defer | ASSUNCAO A4 (alto risco): postura compensatoria documentada; resolver em /gsd-secure-phase (Pitfall 1) — non-root + sandbox/seccomp |
