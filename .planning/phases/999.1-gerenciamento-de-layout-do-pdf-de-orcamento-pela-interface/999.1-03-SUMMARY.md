## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] quotes-pdf-storage.service.test.ts nao tinha mock de PdfTemplatesRepository**
- **Found during:** Task 1 (primeira execucao dos testes)
- **Issue:** O scaffold criado no Plano 01 nao podia prever que o construtor receberia um segundo argumento (PdfTemplatesRepository). O NestJS TestingModule falhou a injecao
- **Fix:** Adicionado `{ provide: PdfTemplatesRepository, useValue: buildMockPdfTemplatesRepository(activeSource) }` ao providers do TestingModule; chamadas `renderHtml` migradas de sincrono para `await` (renderHtml agora e async)
- **Files modified:** quotes-pdf-storage.service.test.ts
- **Commit:** 6ea923d

**2. [Rule 1 - Bug] Tipo Handlebars.KnownHelpers nao exportado pela versao instalada**
- **Found during:** Task 1 (erro TS2694)
- **Issue:** `const KNOWN_HELPERS: Handlebars.KnownHelpers = {}` causou `Namespace 'Handlebars' has no exported member 'KnownHelpers'`
- **Fix:** Tipado como `Record<string, boolean>` que e estruturalmente identico
- **Files modified:** quotes-pdf-storage.service.ts
- **Commit:** 6ea923d

**3. [Rule 1 - Bug] Teste de compile Handlebars precisou ser corrigido (comportamento runtime vs compile)**
- **Found during:** Task 2 (primeiro run RED)
- **Issue:** O teste original esperava que `compile()` lancasse; na realidade `knownHelpersOnly` lanca somente ao chamar o template compilado (`fn({})`)
- **Fix:** Teste refatorado para chamar `fn({})` e assertar que ESSA chamada lanca com mensagem contendo `knownHelpersOnly`
- **Files modified:** quotes-pdf-storage.test.ts
- **Commit:** 6ea923d

**4. [Rule 3 - Blocking] Tasks 1 e 2 implementadas atomicamente (nao em commits separados)**
- **Found during:** Task 2
- **Issue:** D-02 (compile hardened) e D-05 (resolveTemplateSource) compartilham o mesmo metodo `renderHtml`; separar em dois commits criaria um estado intermediario onde os testes da Task 1 passariam mas o servico estaria sem hardening
- **Fix:** Ambas implementadas no mesmo commit `6ea923d` com todos os 19 testes verdes
