## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| .env.local → Next.js build | NEXT_PUBLIC_* são baked into JS bundle — visíveis para qualquer cliente |
| process.env → style inline | EMPRESA_COR_PRIMARIA é injetada num style inline no HTML shell |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-36-01 | Tampering | EMPRESA_COR_PRIMARIA → style inline | accept | Quem controla .env.local já tem acesso total ao servidor; não é vetor de ataque externo. Valor vai para :root { --cor-primaria: VAL } — somente CSS, sem script execution. |
| T-36-02 | Information Disclosure | NEXT_PUBLIC_* no bundle JS | accept | Por design: NEXT_PUBLIC_* são públicas. EMPRESA_CNPJ/NOME/ENDERECO são dados públicos da empresa (CNPJ é registro comercial público). Nenhum segredo é exposto. |
</threat_model>
