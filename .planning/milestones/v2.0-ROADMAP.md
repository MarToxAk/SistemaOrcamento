## Goal

Consolidar o ciclo financeiro interno: hardening do listener de caixa, lançamento de contas a pagar via API REST, upload de comprovantes via SMB, redesign da página de status (Kanban) e dashboard analítico de contas a receber.

## Key Deliverables

1. **AthosListenerService hardened** — reconexão automática com backoff exponencial, notificação Chatwoot ao detectar pagamento no caixa, guard x-internal-api-key em `/api/events/pagamentos`.

2. **API Contas a Pagar completa** — `POST /athos/contas-pagar` insere diretamente no banco Athos, `GET` com filtros por status, `PATCH` para atualização/liquidação. Esquema SQL completo validado com idcontapagar.

3. **Upload de anexos via SMB** — `POST /athos/contas-pagar/:id/anexo` salva no servidor de rede `\\192.168.3.203\html\Anexo\contapagar\{id}\`, valida extensão/tamanho e registra em tabela `anexo`. Integração com Docker Compose via Tailscale.

4. **Kanban de produção** — `/status` redesenhada em 3 colunas (Aprovado/Em Produção/Pronto), cards ricos com badges de pagamento, filtro de carimbo, tabs mobile. Design system BomCusto.

5. **Dashboard de Contas a Receber** — `/contas-receber` com Top Cards BRL, grid de cards por cliente, filtros AVC/VEN/REC/CAN, accordion lazy de títulos individuais, link de navegação no header.

## Key Decisions

| Decision | Phase | Outcome |
|----------|-------|---------|
| Sem DDL em produção no Athos — botão Atualizar manual no dashboard | 27 | Sem riscos de schema drift |
| Top 100 por total_atrasado — sem paginação frontend | 27 | Simples e suficiente para o volume real |
| Route Handlers como proxy (não rewrites no next.config.mjs) | 27 | Padrão existente no codebase mantido |
| Status conta_receber: AVC/VEN (não ABE) | 27 | Compatibilidade com versão específica do Athos |
| FILTER(WHERE) → CASE WHEN (PostgreSQL < 9.4) | 27 | Compatibilidade com versão específica do Athos |
| Upload SMB via pasta montada no Docker (Tailscale) | 25 | Evita dependência de credencial SMB em runtime |
| Swagger documentado para todos os novos endpoints | 24-25 | Operação e debug facilitados |
