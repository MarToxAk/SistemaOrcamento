## Summary

Correção de dois bugs de integração detectados em produção:

1. `getWebhookUrl()` no `EfiService` gerava URL sem o sufixo `/pix`, fazendo a EFI Pay enviar notificações de pagamento para o endpoint errado. Corrigido para retornar `…/integrations/efi/webhook/payment/pix`.

2. `NfseService` usava `??` como fallback de URL SOAP, que não trata string vazia. Com `NFSE_SOAP_URL=` vazio no `.env`, o sistema tentava conectar em `?wsdl` (path inválido) causando `ENOENT`. Corrigido para `?.trim() || DEFAULT_ENDPOINT`.

Adicionados 3 testes unitários de regressão para `getWebhookUrl()`.
