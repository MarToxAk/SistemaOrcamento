# Summary — 260514-003: Status Page Real-time Dashboard

## O que foi feito

Melhorias na página `/status` (Painel de Produção) baseadas no spec do UPDATE_STATUS_PAGE.md, adaptadas para o stack real (Next.js + Bootstrap + SSE).

## Funcionalidades adicionadas

### 1. SSE em tempo real
- Listener em `/api/events/pagamentos` ativo enquanto a página estiver aberta
- Ao receber evento de pagamento no caixa: exibe toast + auto-refresh da lista
- Diff de status: detecta quando um orçamento mudou de status após refresh e aplica highlight visual (animação verde por 3s na linha)

### 2. Banner persistente de último pagamento
- Banner no topo da página com "Último pagamento no caixa: Pedido #X — HH:MM:SS"
- Estado salvo em `localStorage` → sobrevive a page refresh
- Botão "Fechar" salva no localStorage que o usuário dispensou aquele banner específico

### 3. Indicador claro de pagamento no caixa
- Badge verde "✅ Pago no Caixa #pedido" quando `paidInCashier: true` ou `orderNumber` disponível
- Badge azul "PIX Confirmado" quando pagamento via PIX
- Badge amarelo "Aguardando pagamento" enquanto pendente
- QuoteRow type atualizado com `orderNumber` e `paidInCashier`

### 4. Botões de ação no header
- Botão "Atualizar" manual (força refresh da lista)
- Botão "Novo Orçamento" com link para `/orcamento/novo` preservando query params
