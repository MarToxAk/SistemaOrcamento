# Quick Task 260515-002: Menu de Carimbos na Página /status

## Goal

Adicionar um seletor de filtro para carimbos/badges na página de status, permitindo acompanhar orçamentos por categoria de pagamento/status.

## Contexto

Atualmente a tabela mostra carimbos como:
- "Pago no Caixa" (badge success)
- "PIX Confirmado" (badge primary)
- "Aguardando pagamento" (badge warning)

## Proposta

Adicionar um seletor dropdown na barra de cabeçalho (próximo ao filtro de Status existente):
- Label: "Carimbos:"
- Opções:
  - Todos
  - Pago no Caixa
  - PIX Confirmado
  - Aguardando pagamento

Filtrar a tabela visível baseado na seleção.

## Arquivo

- `apps/frontend/src/app/status/page.tsx`

## Tarefas

1. Adicionar estado `selectedBadgeFilter` com valor padrão "TODOS"
2. Criar mapping de rótulos dos carimbos
3. Adicionar lógica de filtro em `visibleQuotes`
4. Renderizar select de carimbos na barra de cabeçalho
5. Validar com build
