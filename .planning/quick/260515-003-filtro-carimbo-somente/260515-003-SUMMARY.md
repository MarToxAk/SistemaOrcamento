# Summary - 260515-003: Filtro "Somente Carimbos" na Página /status

## O que foi feito

Adicionado um filtro ON/OFF "Somente carimbos" na página /status para exibir apenas orçamentos com pagamento confirmado (Pago no Caixa OU PIX Confirmado), excluindo os em "Aguardando pagamento".

## Arquivo alterado

- `apps/frontend/src/app/status/page.tsx`

## Mudanças implementadas

1. **Adicionadas constantes de tipos de carimbos**:
   - BADGE_TYPES: array com tipos ["PAGO_CAIXA", "PIX_CONFIRMADO", "AGUARDANDO"]
   - BADGE_LABELS: mapeamento de rótulos em português

2. **Removido estado obsoleto**:
   - statusSavingId (não é mais necessário para página read-only)

3. **Adicionados novos estados**:
   - selectedStatusFilter: filtro por status de produção
   - selectedBadgeFilter: filtro por tipo de carimbo
   - onlyWithBadge: toggle para mostrar apenas orçamentos com carimbo

4. **Criadas funções auxiliares**:
   - `getBadgeType(quote)`: determina qual tipo de carimbo (PAGO_CAIXA, PIX_CONFIRMADO, AGUARDANDO)
   - `hasBadge(quote)`: verifica se orçamento tem carimbo (paidInCashier OU paymentConfirmedAt)

5. **Atualizada lógica de filtro** `visibleQuotes`:
   - Filtra por Status E Tipo de Carimbo E se tem carimbo (quando ativo)
   - Usuário pode combinar todos os filtros

6. **Removida funcionalidade de edição de status**:
   - Removida função `handleStatusChange`
   - Removido select de edição da UI
   - Página é 100% read-only para visualização pública

7. **Renderizados novos controles na barra de cabeçalho**:
   - Select "Carimbos": Todos, Pago no Caixa, PIX Confirmado, Aguardando pagamento
   - Checkbox "Somente carimbos": toggle para filtrar

## Validação

- `npm --workspace @bomcusto/frontend run build` ✓ compilou com sucesso
- Sem erros de TypeScript
- Filters funcionam independentemente (Status, Carimbos, Somente Carimbos)
- Combinação de todos os filtros funciona corretamente

## Resultado

Usuário pode agora:
- Filtrar por um status específico (Aprovado, Em produção, Pronto para entrega)
- Filtrar por tipo de carimbo (Pago no Caixa, PIX Confirmado, Aguardando)
- Ativar "Somente carimbos" para ver apenas orçamentos com pagamento confirmado
- Combinar todos os filtros para acompanhamento preciso da produção
